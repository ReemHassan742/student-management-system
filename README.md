# student-management-system


**Project Title:** Student Management System

**Technologies Used:**  
- Django 4  
- HTML5  
- CSS3  
- Bootstrap 5 (Bootswatch theme)

**Description:**  
This web application was built as an exercise to enhance Python and Django skills. It allows users to manage student records by providing functionalities to add, edit, and delete information.

**Key Features:**  
- Add, edit, and delete student records  
- User-friendly interface with a Bootswatch theme  

**Skills Demonstrated:**  
- Backend development using Django  
- Frontend design with HTML5, CSS3, and Bootstrap  
- Database management and CRUD operations
